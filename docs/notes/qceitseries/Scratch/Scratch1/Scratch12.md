## 核心概念
:::tip 核心掌握
1. 顺序结构
    + 理解程序按代码顺序执行的逻辑。
    + 掌握 “移动”“旋转”“说” 等基础指令的组合使用。

2. 循环结构
    + 使用 “重复执行”（无限循环）实现持续动作。
    + 结合条件判断（如 “碰到边缘反弹”）控制循环行为。

3. 事件触发
    + 用 “当绿旗被点击”“当角色被点击” 等事件启动代码。
:::
